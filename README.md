# segmentation-ui
<ul>
  to run application -> <br>
  <li>cd app</li>
  <li>npm i</li>
  <li>cd src</li>
  <li>npm run start</li>
</ul>
<ul>
  to run server -> <br>
  <li>cd backend</li>
  <li>npm i</li>
  <li>cd scripts</li>
  <li>flask --app server run</li>
</ul>
